# Renewable Vents and Geysers

## Purpose

Enable long-term colony planning through cyclical renewable resource sources.

## Source Types

- Gas vents: oxygen, hydrogen, natural gas, steam.
- Liquid geysers: water, polluted water, oil variants.
- Thermal vents: high heat with low material output.

## Cycle Model

- Each source has deterministic seed-derived cycle:
  - `active_duration`
  - `dormant_duration`
  - output rate curve

## Gameplay Role

- Early discovery provides expansion targets.
- Mid/late game infrastructure must buffer dormant periods.
- Encourages storage, automation, and redundancy.

## UX

- Scanner/analysis tool reveals cycle data after observation time.
- UI shows next active window and expected throughput.
