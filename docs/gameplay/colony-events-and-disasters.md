# Colony Events and Disasters

## Purpose

Add periodic pressure spikes that test resilience and planning.

## Event Types

- Surface meteor shower (space layer hazard).
- Heat wave (temporary ambient temperature increase).
- Gas pocket rupture (localized pressure/composition shock).
- Power surge event (wire stress period).

## Design Rules

- Events are telegraphed before impact.
- Events should be survivable with correct preparation.
- Early game event pool stays mild to avoid unavoidable wipes.

## Scaling

- Event severity scales with progression tier and difficulty config.
- Cooldown windows prevent event spam.

## UX

- Pre-event warning with countdown and expected affected zones.
- Post-event report with damage/impact summary.
